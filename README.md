# MarkStar — AI 编程技能集合

一行命令安装 [Claude Code](https://claude.ai/code) 技能，开箱即用。

## 安装

```bash
npx markstar
```

技能会自动安装到 `~/.claude/skills/`，重启 Claude Code 即可使用。

## 技能列表

| 技能 | 命令 | 说明 |
|------|------|------|
| **data-to-chart** | `/data-to-chart` | 智能图表生成，自动选择 ECharts 图表类型并生成可交互 HTML。支持大屏/仪表盘模式。 |

## 卸载

```bash
npx markstar --uninstall
```

## 发布

```bash
npm publish
```
